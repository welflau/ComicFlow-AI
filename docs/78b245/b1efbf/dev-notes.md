## 开发备注
已实现完整的智能画布工作流系统集成测试功能。主要特性包括：

1. **集成测试套件**：
   - 画布渲染测试（初始化、3D场景、视图控制、响应式布局）
   - 节点操作测试（创建、连接、删除、拖拽）
   - 工作流执行测试（创建、数据流、状态管理、错误处理）
   - 实时协作测试（WebSocket连接、多用户同步、冲突解决、状态一致性）

2. **测试数据管理器**：
   - 自动创建测试工作流和节点
   - 测试数据清理和验证
   - 结果统计分析

3. **性能监控模块**：
   - 实时FPS监控
   - 内存使用跟踪
   - 网络延迟测量
   - 性能图表可视化

4. **协作测试模拟器**：
   - 多用户并发操作模拟
   - 冲突场景测试
   - 同步准确性验证

5. **测试报告生成器**：
   - 详细测试报告生成
   - 性能瓶颈识别
   - JSON格式报告导出

6. **服务器端支持**：
   - WebSocket实时通信
   - 测试会话管理
   - 性能指标收集
   - 冲突解决机制

使用方法：
1. 运行 `npm install` 安装依赖
2. 运行 `npm start` 启动服务器
3. 访问 http://localhost:3000
4. 点击"集成测试"按钮打开测试面板
5. 点击"运行测试"开始完整的系统测试
6. 查看实时测试结果和性能指标
7. 导出测试报告进行分析
