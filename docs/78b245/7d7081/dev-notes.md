## 开发备注
实现了完整的连线系统，包括：1. ConnectionSystem - 管理连线的创建、删除、验证和更新；2. ConnectionRenderer - 基于Three.js渲染连线、箭头和数据流动画；3. ConnectionInteraction - 处理拖拽连线、选择连接等交互；4. DataFlowVisualizer - 可视化数据流向和状态。支持从输出端口拖拽到输入端口创建连接，实时预览连线路径，点击连线显示详细信息，自动数据流动画效果，以及节点移动时连线自动更新。
